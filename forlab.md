# 实验相关经验

## 应用使用教程

- **Logisim**
  
  - 打开方式
  ```bash
  cd logisim-workspace
  java -jar logisim-evolution-3.8.0-all.jar 
  ```
- **GTKwave**
  - 使用方式
  ```bash
  \\在对应的文件夹内
  make
  make wave
  ```
- **viavdo**
  - 仿真方式
  ```bash
  \\在对应文件夹
  \\ make clean（如需清除之前错误的测试文件）
  make bitstream
  make vivado
  ```
